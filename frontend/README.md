## Variables de entorno

Crear `frontend/.env` basado en `frontend/.env.example`:

- `VITE_API_BASE_URL` (por defecto `http://localhost:5000`).

## Ejecutar local

1. Instalar dependencias: `npm install` en `frontend/`.
2. Desarrollo: `npm run dev` (http://localhost:5173).
3. Build: `npm run build`.
